malq

a temporary email service. no credentials, just mail.

there's literally 2 fucking endpoints.

GET /api/v1/mail/session - creates a new mail session
 
{
"address": "temp@mail",
"token": "XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX"
}


GET /api/v1/mail/inbox/:token - retrieves emails for the session
 
[
{
"from": "sender@mail",
"to": "temp@mail",
"subject": "Hello World!",
"body": "This is a temporary email service.",
"date": 
},
...
]


note: due to our some of suppliers using websockets, emails are "destroyed" after 2 minutes. womp womp.
